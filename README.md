# Set & print MySQL wait_timeout setting

```console
# ./mysql-wait-timeout.py -h
usage: mysql-wait-timeout.py [-h] [-t TIMEOUT] [-p] [-q]

optional arguments:
  -h, --help            show this help message and exit
  -t TIMEOUT, --timeout TIMEOUT
                        set wait_timeout in seconds
  -p, --print           print wait_timeout in seconds
  -q, --quiet           suppress output
```
